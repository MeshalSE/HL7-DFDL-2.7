# HL7-DFDL-2.7
HL7-DFDL-2.7 is a specialized DFDL schema for IBM App Connect designed to handle HL7 v2.x messages. Optimized for TCP communication, it features native support for the MLLP protocol framing. Note that this is a custom version of IBM’s original DFDL, modified specifically to ensure seamless TCP/MLLP integration for healthcare data flows.
